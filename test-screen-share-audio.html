🎯 屏幕分享音频功能测试

测试新增的系统音频捕获功能 - 现在学生们可以听到纪录片的声音了！

🚀 功能改进说明

  • 使用 getDisplayMedia API 直接获取屏幕分享流
  • 支持系统音频捕获（Chrome 74+）
  • 分别发布视频和音频轨道到 LiveKit
  • 音频使用 music 预设，提供更好的音质
  • 智能检测音频轨道是否成功获取
  • 提供详细的用户反馈和错误处理

🌐 浏览器支持情况

✅ 完全支持系统音频： Chrome 74+, Edge 79+
⚠️ 部分支持： Firefox（需手动选择音频源）
❌ 不支持： Safari（仅支持视频分享）
